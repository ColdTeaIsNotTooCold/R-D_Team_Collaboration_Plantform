# 任务：MVP核心架构搭建 - FastAPI + PostgreSQL + Redis基础框架

## 描述
建立MVP版本的核心架构，包括Python FastAPI后端框架、PostgreSQL数据库、Redis缓存和消息队列的基础配置，为Agent协作平台提供简洁的技术基础。

## 验收标准
- [ ] FastAPI项目结构搭建完成
- [ ] PostgreSQL数据库模型设计（用户、Agent、任务、上下文）
- [ ] Redis配置完成（缓存 + Streams消息队列）
- [ ] 基础API路由和中间件配置
- [ ] Docker Compose本地部署配置
- [ ] 基础的健康检查和监控端点

## 技术细节
- 使用FastAPI + SQLAlchemy + Alembic
- PostgreSQL主表设计：users, agents, tasks, contexts, conversations
- Redis用于缓存和简单的Agent间消息传递
- 简化的认证系统（JWT token）
- 基础的配置管理和环境变量处理
